# Crowd Funding Data Insights Dashboard

# Introduction
This repository contains the code and analysis for the Crowd Funding Data Analysis project. The project aims to analyze crowdfunding campaign data to uncover insights, identify success patterns, and make data-driven decisions for improving campaign performance and funding outcomes. Data analysis and insights were performed using Excel, SQL & Tableau.

# About Project 👨‍💻 :-

For this project, a Crowd Funding dataset was used containing multiple attributes such as category, funding goal, amount raised, number of backers, project status, and timeline.

- Dataset: ~100K+ rows, multiple columns  
- Format: CSV  

Overview  
The Crowd Funding Data Analysis project involves analyzing campaign data to uncover insights such as:

- Project success and failure trends  
- Category-wise performance  
- Backer contribution behavior  
- Funding goal vs success relationship  

The main objective is to build a dashboard with key metrics, insights, and visualizations on project performance, funding trends, success rate, and backer engagement.

Data cleaning is performed using Excel / SQL / Tableau.

• Developed a Tableau Dashboard to perform analysis, producing interactive visualizations to determine project success patterns and provide actionable insights for better campaign strategies.

# Problem Statements

1. What is the overall success rate of crowdfunding projects?  
2. Which categories have the highest number of projects?  
3. Which categories have the highest success rates?  
4. How does funding goal impact project success?  
5. What is the distribution of project outcomes (successful, failed, cancelled)?  
6. Which projects received the highest funding?  
7. Which projects attracted the most backers?  
8. How have project trends changed over time?  
9. What is the relationship between funding goal and success rate?  
10. Which regions contribute the most projects?

# Key Insights 📊

- Around 38% of projects are successful  
- Product Design and Games dominate in project volume  
- Lower funding goals (0–10K) have higher success rates  
- A small number of projects attract most of the funding  
- Many projects fail, indicating strong competition
